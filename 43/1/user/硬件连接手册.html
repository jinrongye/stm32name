4. 模块接线详图

4.1 OLED 显示屏（SSD1306，I2C）

    SSD1306 OLED               STM32F103C8T6
   ┌──────────┐              ┌──────────────┐
   │      GND │──────────────│ GND          │
   │      VCC │──────────────│ 3.3V         │
   │      SCL │──────────────│ PB6          │
   │      SDA │──────────────│ PB7          │
   └──────────┘              └──────────────┘
  • I2C 地址：0x78（8位写地址）/ 0x3C（7位地址）
  • 通信方式：软件模拟 I2C，GPIO 位操作
  • 分辨率：128 × 64 像素

4.2 光敏感应模块

   光敏模块                   STM32F103C8T6
   ┌──────────┐              ┌──────────────┐
   │      GND │──────────────│ GND          │
   │      VCC │──────────────│ 3.3V         │
   │       DO │──────────────│ PA0 (ADC1_CH0)│
   │       AO │   悬空       │              │
   └──────────┘              └──────────────┘
  • DO 引脚：输出模拟电压（0 ~ 3.3V）
  • 遮盖（黑暗）：约 0 ~ 1.5V │ 正常光照：约 1.5 ~ 3.3V
  • 阈值调节：旋转模块上的电位器可改变灵敏度

4.3 HC-SR04 超声波测距模块

    HC-SR04                     STM32F103C8T6
   ┌──────────┐                ┌──────────────┐
   │      GND │────────────────│ GND          │
   │  VCC (5V)│────────────────│ 5V  ⚠       │
   │     TRIG │────────────────│ PB0          │
   │     ECHO │────────────────│ PB1 (TIM3_CH4)│
   └──────────┘                └──────────────┘
  • TRIG：STM32 输出 20μs 高电平脉冲触发测距
  • ECHO：HC-SR04 返回高电平，脉宽与距离成正比
  • 脉宽范围：150μs ~ 25ms（对应 2cm ~ 425cm）
  • 距离公式：距离(cm) = 脉宽(μs) / 58
  • PB1 为 5V 容忍引脚（FT），可直接连接 5V 信号
  • 测量角度：约 15°

6. GPIO 资源占用统计

外设占用引脚复用功能GPIO 模式
ADC1PA0ADC1_CH0模拟输入
TIM3PB1TIM3_CH4浮空输入
GPIOPB0—推挽输出（50MHz）
I2C（软）PB6, PB7—开漏输出
GPIOPC13—推挽输出

剩余可用资源：PA1~PA15（除 PA0）、PB3~PB5、PB8~PB15、PA8~PA15 等仍可用于扩展功能（如接入红外避障、雨滴传感器、SIM900A 等）。

7. 注意事项

  1. HC-SR04 必须 5V 供电 — STM32 的 3.3V 输出无法驱动 HC-SR04，必须使用开发板上的 5V 引脚
  2. PB1 是 5V 容忍（FT）引脚 — ECHO 信号为 5V 电平，PB1 可直接接收，无需电平转换芯片
  3. 所有模块必须共地 — 光敏、OLED、HC-SR04 的 GND 必须与 STM32 的 GND 连接在一起
  4. 光敏阈值可调 — 旋转光敏模块上的电位器可改变灵敏度，使其在"遮盖≈黑暗"时 DO 电压 < 1.5V
  5. 杜邦线接触可靠 — 杜邦线松动可能导致 I2C 通信失败或测距数据异常
  6. 先接线后上电 — 接线完毕并检查无误后再接通电源，避免短路损坏模块
