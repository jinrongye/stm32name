# 夜间智能避障系统 — 硬件连接手册

---

## 1. 硬件模块清单

| 序号 | 模块名称 | 型号/规格 | 数量 | 用途 |
|:----:|----------|-----------|:----:|------|
| 1 | 主控板 | STM32F103C8T6（中容量） | 1 | 系统核心控制器 |
| 2 | OLED显示屏 | SSD1306，128×64，I2C接口 | 1 | 显示工作模式与测距数据 |
| 3 | 光敏感应模块 | 光敏电阻 + LM393比较器 | 1 | 检测环境光照强度，区分日间/夜间 |
| 4 | 超声波测距模块 | HC-SR04 | 1 | 夜间发射超声波并测量障碍物距离 |
| 5 | 板载LED | PC13（蓝色） | 1 | 夜间模式指示灯（低电平亮） |
| 6 | 杜邦线 | 公对母 / 母对母 | 若干 | 模块间连线 |

## 2. 引脚接线总表

| STM32引脚 | 连接至（模块 — 引脚） | 信号类型 | 功能说明 |
|:---------:|:----------------------|:--------:|----------|
| **PA0** | 光敏模块 — DO（数字/模拟输出） | 模拟输入 | ADC1_CH0 读取光敏电压，判断环境明暗 |
| **PB0** | HC-SR04 — TRIG | 推挽输出 | 发送 20μs 高电平触发脉冲，启动一次测距 |
| **PB1** | HC-SR04 — ECHO | 浮空输入 | TIM3_CH4 输入捕获，测量回波高电平脉宽 |
| **PB6** | OLED — SCL | 开漏输出 | 软件模拟 I2C 时钟线 |
| **PB7** | OLED — SDA | 开漏输出 | 软件模拟 I2C 数据线 |
| **PC13** | 板载 LED（低电平有效） | 推挽输出 | 夜间亮起 / 日间熄灭 |

## 3. 电源接线

> **⚠️ 重要：HC-SR04 必须使用 5V 供电！STM32 的 3.3V 输出无法驱动 HC-SR04 正常工作。**

| 电源轨 | 连接至 | 说明 |
|:------:|--------|------|
| **5V** | HC-SR04 — VCC | 超声波传感器供电（不可接 3.3V） |
| **3.3V** | OLED — VCC | OLED 显示屏供电 |
| **3.3V** | 光敏模块 — VCC | 光敏传感器供电 |
| **GND** | HC-SR04 — GND | 共地 |
| **GND** | OLED — GND | 共地 |
| **GND** | 光敏模块 — GND | 共地 |
| **GND** | STM32 — GND | 所有模块与 STM32 共地 |

## 4. 模块接线详图

### 4.1 OLED 显示屏（SSD1306，I2C 接口）

```
    SSD1306 OLED            STM32F103C8T6
   ┌──────────┐           ┌──────────────┐
   │      GND │───────────│ GND          │
   │      VCC │───────────│ 3.3V         │
   │      SCL │───────────│ PB6          │
   │      SDA │───────────│ PB7          │
   └──────────┘           └──────────────┘
```

- **I2C 地址**：0x78（8位写地址）/ 0x3C（7位地址）
- **通信方式**：软件模拟 I2C，GPIO 位操作
- **分辨率**：128 × 64 像素

### 4.2 光敏感应模块

```
   光敏模块                   STM32F103C8T6
   ┌──────────┐           ┌──────────────┐
   │      GND │───────────│ GND          │
   │      VCC │───────────│ 3.3V         │
   │       DO │───────────│ PA0 (ADC1_CH0)│
   │       AO │   悬空    │              │
   └──────────┘           └──────────────┘
```

- **DO 引脚**：输出模拟电压（0 ~ 3.3V）
  - 遮盖（黑暗）：约 0 ~ 1.5V
  - 正常光照：约 1.5 ~ 3.3V
- **阈值调节**：旋转模块上的电位器可改变灵敏度
- **AO 引脚**：本设计不使用（悬空即可）

### 4.3 HC-SR04 超声波测距模块

```
    HC-SR04                     STM32F103C8T6
   ┌──────────┐              ┌──────────────┐
   │      GND │──────────────│ GND          │
   │      VCC │──────────────│ 5V  ⚠️       │
   │     TRIG │──────────────│ PB0          │
   │     ECHO │──────────────│ PB1 (TIM3_CH4)│
   └──────────┘              └──────────────┘
```

- **TRIG**：STM32 输出 20μs 高电平脉冲触发测距
- **ECHO**：HC-SR04 返回高电平，脉宽与距离成正比
  - 脉宽范围：150μs ~ 25ms（对应 2cm ~ 425cm）
  - 距离公式：`距离(cm) = 脉宽(μs) / 58`
- **PB1 为 5V 容忍引脚（FT）**，可直接连接 ECHO 的 5V 信号
- **测量角度**：约 15°

## 5. 系统整体接线总览

```
                     ┌─────────────────────────────────────────┐
                     │           STM32F103C8T6                  │
                     │                                         │
    ┌──────────┐     │  PA0 ◄──── 光敏模块 DO     (ADC1_CH0)   │
    │ 光敏模块  │     │                                         │

    └──────────┘     │                                         │
                     │  PB6 ───◄──► OLED SCL       (软件I2C)   │
    ┌──────────┐     │  PB7 ───◄──► OLED SDA       (软件I2C)   │
    │  OLED    │     │                                         │
    │VCC→3.3V │      │  PC13 ─────► LED (板载)     (指示灯)    │
    │GND→GND  │      │                                         │
    └──────────┘     │  5V  ─────── HC-SR04 VCC ⚠️             │
                     │  3.3V ─────  OLED VCC, 光敏 VCC         │
    ┌──────────┐     │  GND ────── 所有模块 GND               │
    │ HC-SR04  │     └─────────────────────────────────────────┘
    │VCC→5V⚠️ │
    │GND→GND  │
    └──────────┘
```

## 6. GPIO 资源占用统计

| 外设 | 占用引脚 | 复用功能 | GPIO 模式 |
|------|:--------:|:--------:|-----------|
| ADC1 | PA0 | ADC1_CH0 | 模拟输入 |
| TIM3 | PB1 | TIM3_CH4 | 浮空输入 |
| GPIO | PB0 | — | 推挽输出（50MHz） |
| I2C(软) | PB6, PB7 | — | 开漏输出 |
| GPIO | PC13 | — | 推挽输出 |

**剩余可用资源**：PA1~PA15（除PA0）、PB3~PB5、PB8~PB15、PA8~PA15 等仍可用于扩展功能。

## 7. 注意事项

| 序号 | 注意项 | 说明 |
|:----:|--------|------|
| 1 | **HC-SR04 必须 5V 供电** | STM32 的 3.3V 无法驱动 HC-SR04，必须使用开发板上的 5V 输出引脚 |
| 2 | **PB1 是 5V 容忍（FT）引脚** | ECHO 信号为 5V 电平，PB1 可直接接收，无需电平转换 |
| 3 | **所有模块必须共地** | 光敏、OLED、HC-SR04 的 GND 必须与 STM32 的 GND 连接在一起 |
| 4 | **光敏阈值可调** | 旋转光敏模块上的电位器可改变灵敏度，使其在"遮盖≈黑暗"时 DO 电压 < 1.5V |
| 5 | **杜邦线接触可靠** | 杜邦线松动可能导致 I2C 通信失败或测距数据异常 |
| 6 | **先接线后上电** | 接线完毕并检查无误后再接通电源，避免短路 |
